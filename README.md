# Project Overview
This project is an interactive Retail Sales Performance Dashboard developed in Microsoft Power BI to analyze sales performance across multiple dimensions including time, geography, and product clusters.
The report combines KPI monitoring, trend analysis, and scenario modeling to support data-driven business decision-making.

# Data
The original source is the Superstore Dataset and can be found [here]([https://www.kaggle.com/datasets/vivek468/superstore-dataset-final](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)).
Although this is the original dataset, some values were changed and tables added (such as Target Table) to increase the scope of the analysis.
The tables can be found in the repository for download.

# Objectives
Analyze retail sales performance across different regions and product segments
Monitor key KPIs such as Revenue, Profit, Margin, and Orders
Evaluate the financial impact of tariff changes through scenario analysis
Enable flexible, dynamic exploration of metrics and dimensions

Key Features
KPI & Trend Analysis

Revenue, Profit, and Margin tracking

Time-based trend analysis

Growth performance evaluation

🔹 Multi-Scenario Tariff Analysis

Simulated tariff impact on profitability

Dynamic adjustment of assumptions

Comparative performance evaluation under different pricing scenarios

🔹 Geography & Product Cluster Analysis

Regional performance breakdown

Product category and cluster comparison

Identification of high- and low-performing segments

🔹 Dynamic Matrix Functionality

One-click switching between:

Dimensions

Metrics

Calculation logic

Parameter-driven analysis within a single report page

Flexible business view without navigating across multiple pages

📈 Business Value

This dashboard demonstrates how structured data modeling and dynamic reporting can:

Improve visibility into retail performance

Support pricing and profitability decisions

Enhance strategic planning through scenario analysis

Enable interactive self-service analytics
